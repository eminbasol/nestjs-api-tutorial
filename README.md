# NestJs REST API tutorial

## Installation
```bash
$ npm install
```
## Running the app

```bash
### Run the API in development mode
#start postgres in docker and push migrations
npm run db:dev:restart
#start api in dev mode
npm run start:dev
```

## Test
```bash
# e2e tests
$ npm run test:e2e

# prisma studio
npx prisma studio
```
